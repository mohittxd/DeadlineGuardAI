# 🛡️ Deadline Guard AI

An urgent task manager and focus assistant with a motivating AI coach to help you beat tight deadlines under pressure — combining short-term deadline management with long-term habit and weekly goal tracking.

## ✨ Features

- **AI Coach Chat** — "LifeSaver AI," a Gemini-powered productivity coach that gives direct, context-aware advice based on your live task list (prioritization, motivation, risk analysis, and planning).
- **Eisenhower Matrix Prioritization** — Tasks are automatically sorted into Do First / Schedule / Delegate / Defer quadrants using AI analysis.
- **Autonomous Task Planner** — Hand the AI a high-level goal and it breaks it down into 3–6 staggered, time-boxed micro-tasks with tactical hints.
- **Focus Timer** — Start/pause/reset countdown sprints tied to individual tasks, with ambient focus soundscapes (Cyber Drone, Warm Focus Ambient, Rain & Storm).
- **Habit & Streak Tracking** — Daily habit checklist with current/longest streak tracking and a forgiving streak-freeze grace mechanic.
- **Weekly Goal Setting** — Checkbox or numeric-target goals scoped to the week, with rollover/archive support for unfinished goals.
- **AI Productivity Analytics** — Weekly summaries, productivity scoring, peak-hours detection, and weak-spot category analysis generated by Gemini.
- **Context-Aware Reminders** — Dynamic, AI-generated nudge messages that escalate in urgency as a deadline approaches (24h / 6h / 1h / overdue).
- **Google Authentication & Cloud Sync** — Sign in with Google (Firebase Auth) and sync tasks, habits, and chat history to a Postgres database.
- **Graceful Offline Fallback** — If no Gemini API key is configured (or a call fails), the app falls back to a rule-based local coaching engine so the experience still works.

## 🧱 Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React 19, TypeScript, Vite, Tailwind CSS, Framer Motion (`motion`), Lucide icons |
| Backend | Express (served via `tsx` in dev, bundled with `esbuild` for prod) |
| AI | Google Gemini (`@google/genai`) with automatic model fallback |
| Auth | Firebase Authentication (Google sign-in) |
| Database | PostgreSQL via Drizzle ORM |
| Hosting target | Node server (`dist/server.cjs`) serving the built Vite SPA |

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- A PostgreSQL database
- A Firebase project (for Google Auth)
- A Gemini API key (optional — app works in offline fallback mode without it)

### Installation

```bash
npm install
```

### Environment Variables

Copy `.env.example` to `.env` and fill in your own values:

```bash
cp .env.example .env
```

| Variable | Description |
|---|---|
| `GEMINI_API_KEY` | Your Google Gemini API key. Without it, the AI coach runs in local offline mode. |
| `APP_URL` | The URL where the app is hosted. |
| `SQL_HOST`, `SQL_DB_NAME`, `SQL_USER`, `SQL_PASSWORD` | PostgreSQL connection details. |

You'll also need to add your Firebase web app config to `firebase-applet-config.json` / `src/firebase.ts`.

### Run Locally

```bash
npm run dev
```

The app will be available at `http://localhost:3000`.

### Build for Production

```bash
npm run build
npm start
```

## 🔒 Security Notes

- `.env` is git-ignored — never commit real API keys or DB credentials. Only `.env.example` (with placeholders) is tracked.
- `firestore.rules` enforces per-user access control in Firestore.
- API routes that touch user data (`/api/sync`) require a valid Firebase ID token via the `requireAuth` middleware.
